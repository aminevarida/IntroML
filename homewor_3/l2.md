# Отчет по лабораторной работе №3
## Описание отчета 
В данной лабораторной работе решается задача бинарной классификации с использованием логистической регрессии. Программа предсказывает вероятность положительного класса на основе предоставленных данных.
## Используемые библиотеки
- **Pandas** - для обработки и анализа данных
- **NumPy** - для математических операций
- **scikit-learn** - для всего  ML-цикла
## Основные характеристики:
- Модель: Логистическая регрессия
- Метрика: ROC-AUC
- Подход:  разделение данных + кросс-валидация
## Исходные данные
- Тренировочные данные: train.csv - содержат целевую переменную y
- Тестовые данные: test.csv - для финального прогнозирования
## Признаки
- Числовые признаки (9):
- age, campaign, pdays, previous emp.var.rate, cons.price.idx, cons.conf.idx, euribor3m, nr.employed
- Категориальные признаки (10):
- job, marital, education, default, housing, loan, contact, month, day_of_week, poutcome
## Модель
Используется логистическая регрессия со следующими параметрами:
- class_weight='balanced' - учет дисбаланса классов
- max_iter=1000 - максимальное количество итераций
- C=1.0 - параметр регуляризации
## Валидация
- Разделение: 80% тренировочные, 20% валидационные
- Кросс-валидация: 5 фолдов (StratifiedKFold)
- Метрика: ROC-AUC
## Выходные данные
Создан файл: my_submission.csv с колонками:
- id - идентификаторы из тестового набора
- y - вероятности положительного класса (от 0 до 1)
Данная прграмма демонстрирует эффективность применения логистической регрессии в сочетании с  предобработкой данных для решения задач бинарной классификации. При работе программы получили ROC-AUC = 0.60460.
